# Buzzy - Professional Beekeeping Services

A modern landing page for Buzzy, a fictitious B2B beekeeping service that helps beekeepers scale their operations profitably. Built with HTML and CSS as part of [The Odin Project](https://www.theodinproject.com/lessons/foundations-landing-page).

## Features

- Responsive flexbox-based layout
- Professional design with warm, earthy color palette
- 4 core service offerings with icons
- Testimonial section with real client story
- Call-to-action section for customer engagement
- Custom Google Fonts (Playfair Display & Outfit)

## Project Focus

This project emphasizes **flexbox layout mastery**, including nested flex containers, responsive grids, and proper alignment techniques. It also demonstrates **UI/UX best practices** for modern B2B landing pages.
